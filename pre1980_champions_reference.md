# Pre-1980 Champions Reference

Source data for the portal's Drought Breakers + (planned) Team drought view.
The portal's `city_index_history.json` already carries COMPLETE per-team title
history from 1980 on; this file is the pre-1980 backfill, sourced from
Wikipedia's canonical champion lists (June 2026) and touchstone-validated.

**Attribution rule (locked with user 2026-06-14):** current-market only.
Titles are keyed by the ERA team name + the market where they were won, so a
relocated franchise's old-city titles do NOT follow it (Minneapolis Lakers
titles stay with Minneapolis, not LA). Renames-in-place DO carry (Washington
Bullets 1978 -> current Washington Wizards, same DC market). All recognized
championships count (pre-Super Bowl NFL, AFL titles 1960-65, BAA, all WS, all
Cups).

Season-year convention for NFL (matches existing PRE_1980_LAST_TITLE): the year
the season STARTED (Super Bowl XIII = 1978 season = Steelers).

Touchstones validated: Yankees 22 WS (thru 1978), Canadiens 21 NHL-era Cups
(thru 1979), Celtics 13 NBA titles (thru 1976).
